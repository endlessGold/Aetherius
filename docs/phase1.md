# Phase 1: 기초 물리 및 환경 역학

## 목표

- Tick 경계에서 결정론적 환경 업데이트가 가능하도록 엔진 코어를 고정한다.
- EnvironmentGrid(Chunk + TypedArray)의 활성 청크 기반 루프를 검증한다.

## 산출물

- 환경 파라미터 레이어별 업데이트 규칙(확산/이류/소스·싱크) 정의
- 결정론 모드(난수/시간/부동소수점 통제) 지점 정의
- 성능 측정 포인트(TPS, activeChunks, totalActiveParameters) 추가

## 검증

- 동일 입력 이벤트 시퀀스에서 동일 결과 재현
- 활성 청크 수 증가에 따른 처리량 측정 및 병목 파악
