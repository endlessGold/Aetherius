# Phase 2: 생물학적 복잡계 및 진화

## 목표

- 목적 생성 GA(Goal-generating GA)를 Tick 루프에 안정적으로 통합한다.
- 생태 상호작용(경쟁/그늘/자원 제한) 모델링 포인트를 확장한다.

## 산출물

- 목적/형질 동시 진화 표현(Genome 스키마, Fitness 분해)
- 공간 분할(Quadtree/Octree) 인터페이스 및 교체 가능 구조
- 개체군 통계/다양성/분포 메트릭

## 검증

- 일정 Tick 후 목적 분포/행동 분포가 유의미하게 분화되는지 확인
- 환경 시나리오(가뭄/풍속 증가/CO2 상승)에 대한 적응 반응 확인
